# Bài thu hoạch "Cloud Mastery 2026 - AI From Scratch"

### Mục Đích Của Sự Kiện

- Giới thiệu cách xây dựng hệ thống AI từ cơ bản đến nâng cao
- Giúp người tham dự hiểu về LLM (Large Language Model) và ứng dụng thực tế
- Cung cấp kiến thức về cách tích hợp AI vào hệ thống và workflow
- Định hướng phát triển trong lĩnh vực AI và Cloud

- **Anh Dinh Le Hoang** - AI Engineer
- **Vinh Banh Cam** - Data Engineer

### Nội Dung Nổi Bật

#### 1. Tổng quan về LLM

- LLM (Large Language Model) là mô hình xử lý ngôn ngữ tự nhiên
- Được huấn luyện với lượng dữ liệu lớn

Có thể triển khai:

- Cloud-based
- Private theo mô hình on-prem hoặc internal system

#### 2. Ứng dụng của LLM

Các use case phổ biến:

- Chatbot
- Automation workflow
- Hệ thống hỗ trợ ra quyết định

Có thể tích hợp vào:

- Web application
- Internal tools
- AI assistant

#### 3. Hệ sinh thái và công cụ

Giới thiệu các framework như:

- LangChain hoặc các framework tương tự

Các khả năng chính:

- Agent orchestration
- Tool calling
- Multi-model support

Agent có thể:

- Đọc và ghi dữ liệu
- Gọi API
- Kết hợp nhiều model khác nhau

#### 4. Agent & Multi-agent System

- Hệ thống AI hiện đại theo hướng agent-based

Hỗ trợ:

- Multi-agent
- Custom workflow
- Khả năng mở rộng cao

Ứng dụng:

- Task automation
- Workflow phức tạp
- AI pipeline

#### 5. Cách xây dựng hệ thống AI

Flow cơ bản:

- Input từ user hoặc system
- Prompt engineering
- Model xử lý
- Output là response hoặc action

#### 6. Prompt Engineering

Các thành phần chính của prompt:

- Role
- Instruction
- Context
- Input data
- Output format
- Examples
- Constraints

Đây là kỹ năng quan trọng khi xây dựng AI application.

#### 7. Kỹ thuật Prompt nâng cao

- Prompt template
- Few-shot hoặc zero-shot
- Structured output (JSON)
- Chain-of-thought reasoning

#### 8. Memory & Context

LLM có thể:

- Giữ context trong session
- Hoặc dùng external memory như database, vector DB

#### 9. Tích hợp thực tế

AI có thể tích hợp vào:

- Browser như Chrome extension
- Workflow hằng ngày

Ví dụ:

- Hỗ trợ viết prompt
- Automation

#### 10. Ứng dụng AI trong thực tế

- Automation công việc
- Hỗ trợ lập trình
- Phân tích dữ liệu
- Sáng tạo nội dung

#### 11. Insight quan trọng

- LLM không chỉ là chatbot mà là nền tảng xây dựng hệ thống
- Prompt có thể xem như một ngôn ngữ lập trình mới
- Agent system là xu hướng phát triển của AI application

AI stack gồm:

- Model
- Prompt
- Tools
- Memory
- Orchestration

### Những Gì Học Được

#### Tư Duy Công Nghệ

- Hiểu rằng AI hiện đại không chỉ là model mà là một hệ thống hoàn chỉnh
- Nhận ra vai trò quan trọng của prompt trong việc điều khiển AI
- Có góc nhìn rõ hơn về cách xây dựng AI application

#### Kiến Thức Kỹ Thuật

Hiểu được:

- LLM hoạt động như thế nào
- Cách xây dựng flow AI cơ bản

Làm quen với:

- Agent-based system
- Prompt engineering
- Memory và context

#### Ứng Dụng Vào Công Việc

Có thể áp dụng:

- Viết prompt hiệu quả hơn
- Sử dụng AI để hỗ trợ coding và học tập

Trong tương lai:

- Xây dựng các AI tools đơn giản
- Tích hợp AI vào project cá nhân

### Trải nghiệm trong event

Sự kiện "Cloud Mastery 2026 - AI From Scratch" giúp làm rõ hơn cách LLM vận hành và cách xây dựng một hệ thống AI từ những thành phần nền tảng.

Các nội dung về prompt engineering và agent-based system đặc biệt hữu ích vì có tính ứng dụng cao trong việc sử dụng AI hằng ngày. Tuy nhiên, một số phần về kiến trúc và framework vẫn còn khá mới nên chủ yếu được tiếp cận ở mức tổng quan.

Nhìn chung, sự kiện mang lại góc nhìn thực tế hơn về AI, không chỉ dừng ở việc sử dụng chatbot mà còn mở rộng sang cách xây dựng và tích hợp AI vào hệ thống.

### Bài học rút ra

- Prompt engineering là kỹ năng quan trọng khi làm việc với AI
- AI application cần kết hợp nhiều thành phần, không chỉ model
- Agent-based system là xu hướng trong tương lai
- Cần học đi đôi với thực hành để hiểu sâu hơn
