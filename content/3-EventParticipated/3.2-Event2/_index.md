# Summary Report: "Cloud Mastery 2026 - AI From Scratch"

### Event Objectives

- Introduce how to build AI systems from basic to advanced levels
- Help attendees understand LLMs (Large Language Models) and their practical applications
- Provide knowledge about integrating AI into systems and workflows
- Give learners direction in the AI and Cloud fields

### Speakers

- **Anh Dinh Le Hoang** - AI Engineer
- **Vinh Banh Cam** - Data Engineer
- **Thinh Nguyen Tuan** - DevOps Engineer

### Key Highlights

#### 1. Overview of LLMs

- LLMs (Large Language Models) are natural language processing models
- They are trained on very large datasets

They can be deployed:

- In the cloud
- Privately through on-premises or internal systems

#### 2. Applications of LLMs

Common use cases:

- Chatbots
- Workflow automation
- Decision support systems

They can be integrated into:

- Web applications
- Internal tools
- AI assistants

#### 3. Ecosystem and tools

Frameworks introduced included:

- LangChain or similar frameworks

Main capabilities:

- Agent orchestration
- Tool calling
- Multi-model support

Agents can:

- Read and write data
- Call APIs
- Combine multiple models

#### 4. Agent and Multi-agent Systems

- Modern AI systems are increasingly agent-based

They support:

- Multi-agent designs
- Custom workflows
- High scalability

Applications:

- Task automation
- Complex workflows
- AI pipelines

#### 5. How to build an AI system

Basic flow:

- Input from users or systems
- Prompt engineering
- Model processing
- Output as a response or action

#### 6. Prompt Engineering

Main components of a prompt:

- Role
- Instruction
- Context
- Input data
- Output format
- Examples
- Constraints

This is an important skill when building AI applications.

#### 7. Advanced prompting techniques

- Prompt templates
- Few-shot or zero-shot prompting
- Structured output in JSON
- Chain-of-thought reasoning

#### 8. Memory and Context

LLMs can:

- Keep context within a session
- Or use external memory such as databases or vector databases

#### 9. Real-world integration

AI can be integrated into:

- Browsers such as Chrome extensions
- Daily workflows

Examples:

- Prompt-writing support
- Automation

#### 10. Practical AI applications

- Work automation
- Coding assistance
- Data analysis
- Content creation

#### 11. Important insights

- LLMs are not just chatbots, but a foundation for building systems
- Prompts can be viewed as a new programming language
- Agent systems are a major trend in AI applications

The AI stack includes:

- Model
- Prompt
- Tools
- Memory
- Orchestration

### Key Learnings

#### Technology Mindset

- I understood that modern AI is not only a model, but a complete system
- I recognized the important role of prompts in controlling AI behavior
- I now have a clearer view of how AI applications are built

#### Technical Knowledge

I understood:

- How LLMs work
- How to build a basic AI flow

I became familiar with:

- Agent-based systems
- Prompt engineering
- Memory and context

#### Applying to Work

Possible applications:

- Writing prompts more effectively
- Using AI to support coding and learning

In the future:

- Building simple AI tools
- Integrating AI into personal projects

### Event Experience

The "Cloud Mastery 2026 - AI From Scratch" event provided a clearer understanding of how LLMs work and how AI systems can be built from foundational components.

The sessions on prompt engineering and agent-based systems were especially valuable because they have direct relevance to everyday AI usage. Some architecture and framework topics were still quite new, so much of the content was taken in at a general level.

Overall, the event offered a more practical perspective on AI, moving beyond chatbot usage toward understanding how AI can be built and integrated into real systems.

### Lessons Learned

- Prompt engineering is an important skill when working with AI
- AI applications require multiple components, not just a model
- Agent-based systems are an important future trend
- Deep understanding comes from combining study with hands-on practice
